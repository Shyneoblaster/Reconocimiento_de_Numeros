# Reconocimiento de Números con CNN y Webcam

Este proyecto implementa un sistema completo para el 
reconocimiento de dígitos escritos a mano (del 0 al 9). 
Utiliza una Red Neuronal Convolucional (CNN) 
entrenada con el dataset MNIST y 
permite reconocer números en tiempo real 
utilizando la cámara web de la computadora.

## Requisitos

Para ejecutar este proyecto, necesitas tener Python instalado y las siguientes bibliotecas:

- tensorflow 
- opencv-python 
- numpy 
- matplotlib
